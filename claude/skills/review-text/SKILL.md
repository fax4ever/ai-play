---
name: review-text
description: Review text for grammar, clarity, formatting, and readability
---

Review the provided text for the following aspects:

## Grammar & Spelling
- Check for grammatical errors
- Identify spelling mistakes
- Flag awkward phrasing

## Clarity & Conciseness
- Identify unclear or ambiguous passages
- Suggest more concise alternatives for verbose sections
- Highlight redundant information

## Markdown Formatting
- Verify proper markdown syntax
- Check heading hierarchy
- Ensure consistent formatting (lists, links, code blocks)
- Validate that links are properly formatted

## Tone & Readability
- Assess if the tone matches the intended audience
- Identify overly complex language that could be simplified
- Check for consistent voice throughout

## Structure & Organization
- Evaluate logical flow of ideas
- Suggest better organization if needed
- Check for proper paragraph breaks

## Output Format
Provide specific, actionable feedback with:
- **Issue:** What the problem is
- **Location:** Where it appears (quote the text)
- **Suggestion:** How to fix it

Present the corrected version in a code block to preserve all markdown formatting.
